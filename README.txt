# Portal de Eventos Comunitarios

Este proyecto es un portal web para gestionar eventos comunitarios. Permite a los usuarios crear, ver y registrarse en eventos de su comunidad.

## Funcionalidades

- **Página de Inicio**: Breve descripción del portal y acceso a eventos destacados.
- **Registro y Autenticación**: Los usuarios pueden registrarse y iniciar sesión.
- **Gestión de Eventos**: Administradores pueden crear, leer, actualizar y eliminar eventos.
- **Interfaz de Usuario**: Diseño responsivo y atractivo usando Bootstrap.
- **Comentarios**: Los usuarios pueden dejar comentarios sobre los eventos.
- **Buscador**: Funcionalidad para buscar eventos por fecha, categoría y ubicación.
- **Sección de Contacto**: Información de contacto de la organización.

## Estructura de Archivos

- `index.html`: Estructura principal de la página web.
- `styles.css`: Estilos de la página usando Bootstrap para diseño responsivo.
- `scripts.js`: Lógica de la aplicación para gestionar usuarios, eventos y comentarios.

## Cómo Usar

1. **Registro**: Crea una cuenta desde la página de registro.
2. **Iniciar Sesión**: Ingresa con tu usuario y contraseña.
3. **Crear Eventos**: Los administradores pueden añadir nuevos eventos.
4. **Ver Eventos**: Todos los usuarios pueden ver y comentar en los eventos.
5. **Buscar Eventos**: Utiliza el buscador para encontrar eventos específicos.
6. **Contacto**: Encuentra información de contacto en la página de contacto.
